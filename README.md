# Configuration for Marlin firmware and Creality Ender 3 (PRO)

### Prerequisite
- Configuration and precompiled firmware is only for BIGTREETECH SKR MINI E3 v2.0 with BL TOUCH
- BL TOUCH is v3.0
- BL TOUCH is connected to Z-STOP on the board
